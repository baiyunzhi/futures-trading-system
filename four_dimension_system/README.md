# 四维三周期客观行情系统

## 目标

只回答三件事：

1. 当下行情正在干什么
2. 什么时候可以下注
3. 怎样下注、哪里失效

## 固定规则

只使用四个维度：

| 维度 | 观察内容 |
|---|---|
| 价格 | 是否突破、跌破、处于区间上沿/下沿/中部 |
| 成交量 | 放量、缩量、量能平稳 |
| 持仓量 | 增仓上涨、增仓下跌、减仓、持仓平稳 |
| 时间 | 波动扩张、波动收缩、节奏平稳 |

只使用三个周期：

```text
周线
日线
小时线
```

不加入任何其他指标。

## 执行流程

```text
1. 周线判断大环境
2. 日线判断当下状态
3. 小时线等待触发
4. 四维度同向确认
5. 先定失效价
6. 再按风险计算手数
7. 触发才下注，不触发只观察
```

## 运行

```bash
cd four_dimension_system
python build_site.py
```

打开：

```text
four_dimension_system/web/index.html
```

## 数据格式

默认会生成一份 `sample_hourly_data.csv`，也可以替换成真实数据。

必须包含字段：

```text
datetime
symbol
open
high
low
close
volume
open_interest
```

## 输出

网页第一屏：

```text
下注计划
```

网页第二屏：

```text
三周期 × 四维度行情矩阵
```

每个品种会给出：

```text
动作：BET / PLAN / WAIT
方向：多 / 空 / 观望
等级：A / B / 观察
触发条件
入场价
止损价
1R目标
2R目标
风险比例
建议手数
失效条件
原因
```

## 下注原则

```text
周线和日线同向，只能进入计划阶段。
小时线也同向，才允许下注。
成交量不支持、持仓量不支持，只观察。
价格没有触发关键位，只观察。
```

## 风险提示

本系统是行情描述和交易计划工具，不构成投资建议。
