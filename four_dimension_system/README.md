# 螺纹钢最近两个月四维行情描述系统

## 目标

只描述行情本身。

标的：

```text
螺纹钢 RB0
```

时间周期：

```text
日线
周线（日线聚合）
```

四个维度：

```text
价格
成交量
持仓量
时间
```

## 核心原则

```text
日线只描述日线自身。
周线由日线聚合，只描述周线自身。
不同周期不互相作为判断依据。
```

## 描述内容

价格：

```text
通过K线开盘、高点、低点、收盘描述行情怎么波动。
```

成交量：

```text
标出最近两个月最大成交量K线，并描述后续行情是否围绕这根K线高低点展开。
```

持仓量：

```text
通过最近两个月首尾持仓量变化描述多空双方参与状态。
```

时间：

```text
使用最近两个月数据窗口，并按日线、周线分别描述。
```

## 运行

```bash
cd four_dimension_system
python build_site.py
```

打开：

```text
four_dimension_system/web/index.html
```

## 数据字段

```text
date
symbol
open
high
low
close
volume
open_interest
```

## 页面输出

```text
1. 日线K线图 + 成交量 + 持仓量
2. 日线客观描述
3. 日线最近两个月数据
4. 周线K线图 + 成交量 + 持仓量
5. 周线客观描述
6. 周线聚合数据
```
