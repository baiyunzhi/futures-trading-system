四维三周期客观行情系统

只使用价格、成交量、持仓量、时间。三个周期：周线、日线、小时线。不加入其他指标。

一步一步的流程

  1. 先看周线：只判断大环境，不下注。
  2. 再看日线：确定当下行情是突破、区间上沿、区间下沿，还是中部震荡。
  3. 最后看小时线：只有小时线触发，才允许执行下注。
  4. 下注前必须确认四个维度：价格方向、成交量主动性、持仓量持续性、时间节奏。
  5. 先确定失效价，再计算手数；没有触发就只做计划，不提前预测。
