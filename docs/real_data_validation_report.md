当前版本已经移除均线和 MACD 对交易决策的影响：

1. 交易信号改为滚动高低点结构突破。
2. 出场改为结构止损、结构止盈、结构转弱/转强、持仓超时。
3. ATR 仍保留，只用于止损、止盈、滑点后风控距离和仓位计算。
4. 页面 K 线图已移除均线和 MACD 子图。
5. 行情描述已移除 MA/MACD/RSI/ADX 指标段。

结论：均线和 MACD 在旧系统中确实参与了交易；移除后当前纯结构版本未达到正收益要求，不能进入模拟自动执行。

## 下一步优化优先级

1. 纯结构规则重建：加入更严格的突破确认、回踩确认和假突破过滤。
2. 环境过滤：2023 年纯结构亏损明显，必须识别不适配环境。
3. 出场逻辑：当前结构转弱/转强出场仍偏粗糙，需要按品种优化。
4. 组合逻辑：对 RB0、C0、Y0、AL0 等拖累品种加入滚动准入强化。
5. 验证逻辑：继续加入训练期、验证期、样本外期和滚动回测。
