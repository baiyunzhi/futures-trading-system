## 诊断结论

1. 如果某一年明显亏损，说明系统对市场环境切换不稳定。
2. 如果利润集中在少数品种，说明策略不是全品种稳健优势。
3. 如果盈亏比低于 1.2，优先优化出场和过滤，不优先提高交易频率。
4. 如果最大回撤大于收益，系统不能进入模拟自动执行。
